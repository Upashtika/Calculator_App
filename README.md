# Calculator App

A simple and modern Calculator Android App built using Android Studio and Java/XML.  
This app performs basic arithmetic operations with a clean and responsive UI.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Modern dark UI
- Responsive layout

## Technologies Used

- Java
- XML
- Android Studio
